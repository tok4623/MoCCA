This is the code repository for MoCCA

## Run the tests
To run the scripts used to generate the Paper:
```bash
cd /root/of/git/repo
python3 main.py 
# or if you want to run just one sceanrio
python3 tests/scenarioA.py # or tests/scenarioB.py
```
